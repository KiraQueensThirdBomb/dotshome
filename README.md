<div align="center">
  <h1>Install</h1>
<h3>Please back up your configs before installation since IT WILL DELETE all your configs for your apps</h3>
</div>

clone this page with git
```shell
git clone https://github.com/KiraQueensThirdBomb/dotshome.git
```
  
prep the config file
```shell
cd ~/dotshome
chmod +x config.sh
```
run the config
```shell
~/dotshome/config.sh
```
stuff that the script for some reasons didn't do
```shell
chmod +x ~/.config/rofi/launchers/type-2/launcher.sh
chmod +x ~/.config/rofi/powermenu/type-2/powermenu.sh
chmod +x ~/.config/waybar/scripts/reload.sh
```
after finishing you can to reboot into hyprland and use this command to create the color schemes with the wallpapers
```shell
matugen image ~/Wallpapers/wall1.jpg
```
and with that you should be set and ready to go<br>
also please note that thease dots come with departuremono nerd font 3.4.0<br>
