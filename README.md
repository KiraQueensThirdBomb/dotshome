<div align="center">
  <h1>Install</h1>
</div>
<details><summary>Click to expand</summary>

clone this page with git
```shell
git clone https://github.com/KiraQueensThirdBomb/dotshome.git
```
  
prep the config file
```shell
cd ~/dotshome
chmod +x config.sh
```
run the config
```shell
~/dotshome/config.sh
```
